[uv](https://docs.astral.sh/uv/) looks like a very promising new python package manager. Its fast, does global python tool installs, and has sensible defaults.

The only downside against conda is that uv only installs python tools, while conda can install other apps and tools.

2026: uv has rightly taken over the python world and [openai acquired uv](https://simonwillison.net/2026/Mar/19/openai-acquiring-astral/).
