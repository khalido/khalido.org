## Colophon

This website has gone through many iterations - my original blog was html pages on geocities, than movabletype, wordpress, my own python script, then hugo, then back to [an overly complicated python script](https://github.com/khalido/blog/blob/master/bloggy/blog.py), then finally I switched to astrojs - by hand!
